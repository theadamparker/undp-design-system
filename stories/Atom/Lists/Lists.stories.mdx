# List component
List component show a number of connected items written consecutively, typically one below the other
.

<div className="subheading">Unordered List</div>
<p>An unordered list starts with the <code>ul</code>tag. Each list item starts with the <code>li</code> tag. The list items will be marked with bullets (small black circles) by default.</p>

<Story name="Unordertag">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <Unorderlist data={caption}></Unorderlist>;
  }}
</Story>

<div className="subheading">Ordered List</div>
<p>An ordered list starts with the <code>ol</code> tag. Each list item starts with the <code>li</code> tag. The list items will be marked with numbers by default.</p>

<Story name="Ordertag">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <Orderlist data={caption}></Orderlist>;
  }}
</Story>

<div className="subheading">Description List</div>
<p>A Description list is a list of terms and corresponding Descriptions. Description lists are typically formatted with the term on the left with the description following on the right or on the next line.</p>

<Story name="Descriptiontag">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <Descriptionlist data={caption}></Descriptionlist>;
  }}
</Story>
