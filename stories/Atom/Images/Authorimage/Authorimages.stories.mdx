# Author Image Component
The Author Image Component is used to embed an image in a web page.

<ul>
<li>Author Images are saved in <code>assets/images/</code> folder in SVG, PNG, JPG, JPEG format. </li>
<li>The <code>img</code> tag has two required attributes:</li>
  <ul>
    <li>src - Specifies the path to the image</li>
    <li>alt - Specifies an alternate text for the image</li>
  </ul>
</ul>
