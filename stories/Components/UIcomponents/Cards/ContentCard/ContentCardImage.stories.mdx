# Single Content Card Component
The Single Content Card component defines a card of information data with some items - Title, Image, Paragraph, and CTA Link. We are using the grid system here <code>assets/scss/grid.scss</code>. Atoms tags are used for creating this card.

### Single Content Card
Single content card with image are show the Title, Image, and CTA Link. The background color is different on hover.

<Story name="With Image">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return (
      <ContentCardImage
        data={caption.contentdata} {...args}
      ></ContentCardImage>
    );
  }}
</Story>

- Take HTML from the HTML tab in the canvas

- Component SCSS file: <code>singlecontent.scss</code>
- <u>To use CSS include the following files in your implementation</u>
- The compiled CSS file from <code>assets/css/style.css</code>
