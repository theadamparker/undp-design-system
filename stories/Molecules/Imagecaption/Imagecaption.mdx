# Image Caption + Credit Component
Image Caption + Credit component used for the image caption include: paragraph and image author name.

<Story name="Imagecaption">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <Imagecaption label={caption.detail} name={caption.name} paragraph={caption.paragraph}></Imagecaption>;
  }}
</Story>
