# Tertiary Bloack

Tertiary Tag Block defines a section of information data with some items - Location, Country, Description. <code>H2</code> <code>H4</code> <code>P</code> Atoms tags are used for creating this section.

<Story name="Tertiary Block">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <Tertiary headerText={caption.header} countryText={caption.country}  descriptionText={caption.description}></Tertiary>;
  }}
</Story>
