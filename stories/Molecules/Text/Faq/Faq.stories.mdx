# Faq Component
The FAQ component is responsible for managing the isOpen and toggle functionality to each Item component define in this component, for display the data used items fileds - Header title, Description.  <code>H5</code> <code>P</code> Atoms tags are used for creating this section.

<Story name="Faq">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <Faq headerText={caption.header} descriptionText={caption.description}></Faq>;
  }}
</Story>

<div className="subheading">Dependency</div>

Libraries or another dependency which are used in the component.

<ul>

<li>Custom JS file (Faq.js) is located in the same component folder.</li>
<li>ID - (#accordion) is used in the JSX file to connect the Custom JS file to achieve the functionality.</li>
</ul>
